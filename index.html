Knockdown - A Javscript Dice Game

  • Project : Knockdown
  • Client : Hobby Project
  • Technologies : html, javascript, CSS Flexbox, Javascript Math object
  • Description :

    When I learned about the math object and how to generate a random number in a range I realized I could simulate the roll of a die.

    So I started with creating a div for each face of the die and built the face's dots with CSS flexbox. No images are used everything is made with HTML5 and CSS Flexbox.

    Then I created a function that displayed what number of dots on the face that I passed into it. Then I created a function to simulate the roll of the die.

    I reached that point and realized that I could write some loops and conditions to create a dice game, and thus it evolved.

Star Wars Phrase Hunter

  • Project : Techdegreee Project
  • Technologies : html, oop javascript, Javscript dom manipulation, css
  • About :

    This was my fourth project at Treehouse for my TechDegree. This unit project shows my ability to used object oriented programming principles to create a browser-based javascript game.

    Each game starts off with a random star wars quote that is placed on the gameboard for the player to guess by filling in the blanks. the player only gets 5 lives which is designated by the blue hearts. When a wrong letter is guessed a live is lost (shown by hearts turning grey). When the letter is pressed the onscreen button becomes disabled and the player can no longer lose a life for pressing it. When the lives are all lost or the phrase is guessed the game is over.

    For extra credit I enabled the real keyboard to function and made the game star wars themed.

Startup Employee Directory

  • Project : Techdegreee Project
  • Technologies : html, fetch api, random user generator api, ajax, json, css
  • About :

    This was my fifth project at Treehouse for my TechDegree. This project highlights my ability to create applications that communicate with APIs and work with microservices and databases to build useful tools and relevant infomation quickly and easily.

    This is an app for a fictional company called Awesome Startup, a distributed company with remote employees working all over the world. They need a smart way for employees to share contact information with each other.

    For extra credit I enabled the real keyboard to function and made the game star wars themed.

    Using the Random User Generator API (https://randomuser.me/) to grab information for 12 random "employees" and use that data to build a prototype for an Awesome Startup employee directory.

    Using the fetch API i request a JSON object from the Random User Generator API and parse the data so that 12 employees are listed in a grib with thier thumbnail image, full name, email, and location. Clicking the employee's image or name opens a modal window with more detailed information such as employee's birthday and address.

    The functionality of the gallery and modal window was required to be limited to vanilla javascript and jquery. This is to illustrate that I have knowledge of how things run under the hood of the popular libraries and frameworks available today.
