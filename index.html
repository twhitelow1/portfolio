Knockdown - A Javscript Dice Game

  • Project : Knockdown
  • Client : Hobby Project
  • Technologies : html, javascript, CSS Flexbox, Javascript Math object
  • Description : When I learned about the math object and how to generate a random number in a range I realized I could simulate the roll of a die. So I started with creating a div for each face of the die and built the face's dots with CSS flexbox. No images are used everything is made with HTML5 and CSS Flexbox. Then I created a function that displayed what number of dots on the face that I passed into it. Then I created a function to simulate the roll of the die. I reached that point and realized that I could write some loops and conditions to create a dice game, and thus it evolved.
